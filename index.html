Android APK Guide
📱 Build Android APK (5 steps): 1. Install Android Studio
Download from developer.android.com/studio

2. New Project
New Project → Empty Views Activity → Java → Min SDK 21

3. Add HTML file
In Project view (switch from "Android" to "Project" at top left)
Navigate to: app → src → main
Right-click main → New → Directory → type assets → Enter
Copy this HTML file into that folder → rename to app.html

4. activity_main.xml (replace all content)
<WebView android:id="@+id/webView"
android:layout_width="match_parent"
android:layout_height="match_parent"/>
5. MainActivity.java (inside onCreate, after setContentView)
WebView web = findViewById(R.id.webView);
WebSettings s = web.getSettings();
s.setJavaScriptEnabled(true);
s.setDomStorageEnabled(true);
s.setAllowFileAccessFromFileURLs(true);
s.setAllowUniversalAccessFromFileURLs(true);
web.setWebChromeClient(new WebChromeClient());
web.loadUrl("file:///android_asset/app.html");
AndroidManifest.xml — add inside <manifest>:
<uses-permission android:name="android.permission.INTERNET"/>
Inside <application> tag add:
android:usesCleartextTraffic="true"
Build APK:
Build → Build Bundle(s)/APK(s) → Build APK
File appears at: app/build/outputs/apk/debug/app-debug.apk
Transfer to phone → Install (enable "Install unknown apps" first)
☁️ Cloud Sync (Firebase)
1. Go to console.firebase.google.com
2. Create project → Add Web App
3. Copy config object
4. In this HTML file find FIREBASE_CONFIG_HERE
5. Replace with your config → Save & reload
