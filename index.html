Android APK Guide
📱 Full Android Build: 1. Install Android Studio
Download: developer.android.com/studio

2. New Project
New Project → Empty Views Activity → Java → Min SDK 21

3. Add HTML file
Switch panel to "Project" view → app → src → main
Right-click main → New → Directory → type: assets
Copy this HTML into assets folder → rename to app.html

4. activity_main.xml
<WebView xmlns:android="http://schemas.android.com/apk/res/android" android:id="@+id/webView" android:layout_width="match_parent" android:layout_height="match_parent"/>
5. MainActivity.java (inside onCreate after setContentView)
WebView web = findViewById(R.id.webView); WebSettings s = web.getSettings(); s.setJavaScriptEnabled(true); s.setDomStorageEnabled(true); s.setAllowFileAccessFromFileURLs(true); s.setAllowUniversalAccessFromFileURLs(true); web.setWebChromeClient(new WebChromeClient()); web.setWebViewClient(new WebViewClient(){ @Override public boolean shouldOverrideUrlLoading( WebView v, WebResourceRequest r){ String url = r.getUrl().toString(); if(url.startsWith("https://wa.me")|| url.startsWith("whatsapp:")){ startActivity(new Intent( Intent.ACTION_VIEW, Uri.parse(url))); return true; } if(!url.startsWith("file://")&& !url.startsWith("about:")){ startActivity(new Intent( Intent.ACTION_VIEW, Uri.parse(url))); return true; } return false; } }); web.loadUrl("file:///android_asset/app.html");
Add these imports at top of MainActivity.java:
import android.webkit.*; import android.content.Intent; import android.net.Uri;
6. AndroidManifest.xml — add before <application>:
<uses-permission android:name="android.permission.INTERNET"/>
Inside <application> tag:
android:usesCleartextTraffic="true"
7. Build APK
Build → Build Bundle(s)/APK(s) → Build APK
APK: app/build/outputs/apk/debug/app-debug.apk
